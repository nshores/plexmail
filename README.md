# plexmail
Python tool to email plex shared users

**Usage**

**Edit the following variables in main.py to fit your setup**

 	

~~~~
#Plex Setup
plex_username = 'xxxx'
plex_password = 'xxxxx'
#End Plex Setup

#SMTP Setup
email_username = 'xxxx'
email_password = 'xxxxx'
email_host = 'smtp.office365.com'
email_from = 'xxxx'
email_port = '587'
email_body = 'Nothing to see here yet'
#End SMTP Setup

 	

~~~~

Run with python3.x

    python3 main.py
